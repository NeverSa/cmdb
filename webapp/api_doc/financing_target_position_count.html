按位置统计标的数量接口
  • 接口地址

    /api/financing_target_position_count.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
  • 响应结果
    名称 类型 一定存在 示例 说明
    APP_SELLOUT_LIST int 否 100 售完列表的数量
    APP_REPAYMENT_LIST int 否 已还款列表

    注意:这些是根据标的位置统计出来的,意味着,如果某个标的位置不存在标地,那么这位置将什么也不返回,不存在这个返回值中

  • 示例
    {
    "APP_HOME": 1,
    "APP_ZONE_CPB": 42,
    "WX_ZONE_CPB": 33,
    "APP_SELLOUT_LIST": 1,
    "WX_HOME": 1,
    "APP_ZONE_SCB": 1,
    "WX_SCB": 1
}
