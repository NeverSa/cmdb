重置登录密码接口
  • 接口地址

    /api/reset_loginpwd.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    mobile string 是 手机号码/也是登录名
    newPwd string 是 新登录密码(md5加密)
    verifyCode string 是 短信验证码 (目前随便填)
  • 响应结果
    名称 类型 一定存在 示例 说明
    code string 是 SUCCESS SUCCESS : 重置密码成功
    VERIFY_CODE_ERROR : 验证码错误
    USER_NOT_EXIST : 用户不存在
    PARAM_ERROR:参数错误
  • 示例
    {							
	code:"SUCCESS" 
}
