获得理财顾问列表接口
  • 接口地址

    /api/financial_advisor_list.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
  • 响应结果
    提示:该接口返回JSON数据,以下是数组中元素的描述.有些值比如:gmtCreate、gmtWrite等没有业务意义就不描述了
    名称 类型 一定存在 示例 说明
    id string 是 编号,一般是guid
    name string 是 光头强 姓名
    mobile string 是 手机号码
    headPic string 是 一个头像URL
    为了扩展性强一点
    这个地方有可能是gif
    companyName string 杭州满溢科技有限公司 公司名字
    staff string 高级理才顾问 职位名称
    experience string 3年理才从业经验 从业经验描述
