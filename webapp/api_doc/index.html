接口文档使用说明
  • 什么人需要阅读该文档?

    本接口文档是为了帮助app开发人员与后台服务器交互而开发,因此android、ios需要阅读本文档.

  • 接口的形式

    采用 http请求的方式向客户端或外部应用提供服务,默认请况下所有的请求即可以支持GET也可以支持POST(特殊情况下会在接口描述中特别标注.).大多数接口都会有一个 token的参数,该参数是为了适别请求人的身份票证,因些我们在调用这些业务接口之前需要调用一个token获取接口,并保持票证的有效性.另外在接口地址描述的时候都只是用了相对地址,而前面的主机地址部分即可以是测试环境,也可以是正式环境.但测试的时候请都在测试环境下完成.

    所有接口都采用POST提交

  • 支持的环境
    环境名称 类别 地址 说明
    测试环境 接口 https://test.laicunba.com/{接口相对地址} 用于app开发与测试阶段的调试工作
    温馨提示:由于会不定时的发布,有可能会出现中断服务的情况,一般30秒钟后正常 双向认证
    测试环境 H5 http://test.laicunba.com/{h5相对地址} 无认证
    正式环境 接口 https://api.laicunba.com/{接口相对地址} 双向认证
    正式环境 H5 https://h5.laicunba.com/{H5相对地址} ca机构单向认证

  • 所有请求公共参数
    名称 类型 说明
    _channel string 用户后台分析这些请求来自哪个渠道,根据运营人员的要求相应版本编入对应值 支持运营用
    _app_version string android版本请使用数值版本号
    _app_type string android/ios
    _device string 设备型号
    _device_os string 手机操作系统
    app_token string 有则传,没有则不传,不考虑token的有效性 仅限H5页面
  • H5页面如何跳转到APP界面?

    如果碰到在h5中需要点某个按钮或是链接,请在参数列表加上以下参数. webView控件会拦截请求地址,然后根据相应的参数跳对应的原生界面!

    例: http://xxx.laicunba.com/xxx.htm?app_view=home
    app界面 参数
    首页 app_view=home
    理财专区 app_view=zone
    我的账户 app_view=acc
    随存宝 app_view=scb
    随存宝提现完成 app_view=scb_withdraw_finish
    标的详情 app_view=detail&id=XXXXXXXXX
    未登录则登录/如果已登录则不管/登录后跳回该页面,并且传回app_token=XXXXX参数 app_view=login
    弹出分享框 app_view=share&title=分享的标题&url=分享的地址&summary=分享的摘要
    该页面需要分享,如果发现地址中有这些参数,则表示该页面需要有右上角的分享按钮. app_view=has_share_btn&title=分享的标题&url=分享的地址&summary=分享的摘要
    注册页面 app_view=regist
    发现页面 app_view=discovery
    注意:这些请求的地址域名需要为我们的来存吧域名
