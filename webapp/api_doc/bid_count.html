投资统计接口
  • 接口地址

    /api/bid_count.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
  • 响应结果
    名称 类型 一定存在 示例 说明
    bid_amount string 是 1,000,000.00 用户累计投资(带格式金额)
    bid_profit string 是 50,000.00 用户累计赚取(带格式金额)
    bid_user int 是 10000 投资用户数
  • 示例
    {							
	bid_amount:"1,000,000.00",
	bid_profit:"50,000.00"
}
