获得用户信息接口
  • 接口说明

    通过该接口在登录成功后,获取用户的信息,以便展现如:身份证,真实姓名等等

  • 接口地址

    /api/get_user_profile.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    token string 是 身份识别信息,参考 "获得用户Token接口"
  • 响应结果
    名称 类型 一定存在 示例 说明
    code string 是 SUCCESS SUCCESS : 获得用户成功
    (只有这个值,才有其它内容)
    TOKEN_ERROR : token无效
    id string 3e804a7d-50e7-49db-a3cd-514dbce7dbdb 唯一标识
    gmtRegist long 1452231541516 注册时间 (毫秒数)
    mobile string 18367156949 手机号码
    trueName string 张三 姓名
    绑定银行卡后才会存在
    pid string 身份证号码
    绑定银行卡后才会存在
    inviteCode string 是 888888 我的邀请码
  • 示例
    {
    "gmtRegist": 1452231541516,
    "id": "16827d32-2be6-4683-85a0-78331a26fcfc",
    "mobile": "1",
    "pid": "352523191108241222",
    "trueName": "张三",
    "inviteCode": "888888",
    "code": "SUCCESS"
}
