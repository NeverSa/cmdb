获取图片接口
  • 接口地址

    /api/get_image_list.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    position string 是 WX_HOME banner图位置
  • 响应结果
    名称 类型 一定存在 示例 说明
    id string 是 c47c4a18-4a8f-48ad-b094-a89f3af6d34a 唯一标识
    title string 否 来存吧-新年狂欢 标题
    imageUrl string 是 http://www.laicunba.com/upload/images
    /20160107/1438580268158.png    		 图片地址
    linkUrl string 是 http://www.laicunba.com 链接地址
    sortNum int 是 1 排序编号
    startDate long 是 1451577600000 展现开始时间（毫秒数）
    endDate long 是 1454255999000 展现结束时间（毫秒数）
