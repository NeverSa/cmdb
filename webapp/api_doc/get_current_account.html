查看活期账户接口(新)
  • 接口地址

    /api/get_current_account.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    token string 是 身份识别信息,参考 "获得用户Token接口"
  • 响应结果
    名称 类型 一定存在 示例 说明
    status string 是 ENABLE ENABLE : 正常使用
    DISABLE : 未激活,弹出激活页面
    balance money 1000.00 持有金额
    annualizedRates float 7.0 年化利率
    tenThousandDayProfit money 1.92 万元收益/日
    yesterdayProfit money 0 昨日收益
    totalProfit money 0 累计收益
    financingTargetId string 249d21f0-ae24-431f-813c-96c6676f97f6 标的ID,判断为空或不存在的情况
  • 示例
    {
    "balance": 0,
    "totalProfit": 0,
    "annualizedRates": 7,
    "tenThousandDayProfit": 1.92,
    "yesterdayProfit": 0,
    "status": "DISABLE"
}
