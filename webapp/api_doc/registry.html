注册表查询接口
  • 接口说明

    您可以通过传入一个或多个key,它会返回一组相应的数据,该接口应用非常广泛,比如:客户电话,服务时间,说明性的文字,等等

  • 接口地址

    /api/registry.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    key string 可以传入多个key比如: key=1&key=2
  • 可用Key列表
    Key 例 说明
    sys.service.mobile 客户电话
    sys.service.mobile.time 客户电话的服务时间
    sys.wx 微信公众号
    sys.qq QQ交流群
    sys.sina.weibo 新浪微搏
    sys.service.email 客服邮箱
    sys.pic.domain 图片服务器域名
    sys.app.home.banner.v2

    • 标的详情 pic action="financing_target_detail" financing_target_id

  • 响应结果
    该结果为一个map形式的json数据,key为您传的值,value为系统中对应该key的值
  • 示例
    {"sys.service.mobile":"400-88888","sys.service.mobile.time":"每日9:00-21:00"}
