通过位置查找标的接口
  • 接口说明

    该接口可用于特定位置的标的列表显示,比如APP首页随存取

  • 接口地址

    /api/financing_target_findby_position.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    position string 是 位置标记
    WX_SCB : 微信端随存宝
    APP_HOME : app首页
    APP_ZONE_SCB : app理财专区,随存宝
    APP_ZONE_CPB : app理财专区,存票宝系列
    APP_SELLOUT_LIST : app售完列表
    APP_REPAYMENT_LIST : app已还款列表
    limit int 1 还回条数
  • 响应结果
    提示:该接口返回JSON数组,以下是数组中元素的描述.有些值比如:gmtCreate、gmtWrite等没有业务意义就不描述了
    名称 类型 一定存在 示例 说明
    id string 是 8ae94a13-3d2b-4246
    -9687-55402dc1917b    		 标的ID
    name string 是 随存取0001 标的名称
    type string 是 SCB SCB: 随存宝,CPB:存票宝
    code string 标的代码
    collectAmount money 是 50000 计划融资金额(元)
    finishAmount money 是 1000 完成融资金额(元)
    注:存在
    finishAmount > collectAmount 的可能性
    period int 是 7 期限(天)
    annualizedRates float 是 9.5 年化利率(单位:百分之)
    minAmount money 100 起投金额(元)
    limitAmount money 100000 个人上限
    status string 是 new : 新创建
    onsale: 可购买的
    offsale:已下架
    closed : 关闭的
    checking : 核对中
    archive:存档
    beginDate long 1452040129000 开标时间(毫秒数)
    endDate long 1454200136000 结标时间(毫秒数)
    bidCount int 是 99 投标次数
    pics string 质押标的物照片(json格式) 注:json里面 path是图片的相对路径,通过注册表查询接口(key:sys.pic.domain)获取图片域名
    picsDescription string 质押物说明,html格式
    description string 项目介绍
    cityAddress string 否 浙江 杭州 城市地址
    tagetMark string 否 hot 标签（hot：HOT，activity：活动,）
    cashMark string 由XXX兑付 兑付承诺说明
    说明:如果返回空,则使用 "注册表查询接口" key:sys.cash.mark,获取
    interestWay string T(成交日) +1 起息时间
    incrRate money 0.5 加息额度 注意:不管该字段是否有值,真正的利息都以annualizedRates计算,app上基础利息应该是 annualizedRates - incrRate (如果存在的话)
  • 标的状态判定规则
    注:时间条件,都是当时间值存在的情况下
    是否可买条件: status = "onsale" 并且当前时间 > beginDate 并且当前时间 < endDate
    售罄条件: finishAmount >= collectAmount
    已下架(情况一)条件: status = "offsale"
    已下架(情况二)条件: status = "onsale" 并且当前时间 > endDate
    即将开售条件: status = "onsale" 并且当前时间 < beginDate
