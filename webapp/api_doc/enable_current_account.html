(新活期)开通活期账户接口
  • 接口地址

    /api/enable_current_account.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    token string 是 身份识别信息,参考 "获得用户Token接口"
  • 响应结果
    名称 类型 一定存在 示例 说明
    code string 是 SUCCESS SUCCESS : 激活成功,多次调用都只会返回这个
    (只有这个值,才有其它内容)
    TOKEN_ERROR : token无效
    ERROR : 激活失败
  • 示例
    {							
	code:"SUCCESS" 
}
