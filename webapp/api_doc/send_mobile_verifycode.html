发送手机短信验证码接口
  • 接口说明

    该接口用于各种短信验证码发送的地方,由于发送短信可能比较慢,所以建议在发送短信的地方提示用户,进行倒计时或是超时可重发提示. 如果发送后很久都没收到,有可能是短信服务商不稳定(请及时联系我们以便跟踪原因)

  • 接口地址

    /api/send_mobile_verifycode.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    mobile string 是 接收验证码的手机号码
    verifyType string 是 验证码类型
    REGIST:注册验证码
    RESET_LOGINPWD:重置登录密码
    RESET_PAYPWD: 重置支付密码
  • 响应结果
    名称 类型 一定存在 示例 说明
    code string 是 SUCCESS SUCCESS:发送成功(但不保证是否送达)
    MOBILE_ERROR : 手机号不正确(一般是指格式不正确)
    PARAM_ERROR:参数错误
    USER_EXIST:用户已存在
    VERIFY_TYPE_ERROR:不支持的验证码类型
    USER_NOT_EXIST_OR_DISABLE:用户不存在或被禁用
    MSG_AMOUNT_LIMITED: 超出发送次数限制1小时内同一ip可以发100条短信,12小时内同一手机可以发5条短信
  • 示例
    {							
	code:"SUCCESS" 
}
