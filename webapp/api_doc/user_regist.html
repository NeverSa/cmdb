用户(投资者)注册接口
  • 接口地址

    /api/user_regist.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    mobile string 是 手机号码/也是登录名
    loginPwd string 是 md5加密后过的登录密码
    verifyCode string 是 短信验证码
    目前发送短信的功能暂未完成,默认验证码为0000
    inviteFrom string 邀请码
  • 响应结果
    名称 类型 一定存在 示例 说明
    code string 是 SUCCESS SUCCESS : 注册成功
    USER_EXIST : 用户已存在
    VERIFY_CODE_ERROR : 验证码错误
    MOBILE_ERROR:手机号码错误
    PARAM_ERROR:参数错误
  • 示例
    {							
	code:"SUCCESS"
}
