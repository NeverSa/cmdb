获得用户存票宝的收益概况接口
  • 接口地址

    /api/get_cun_piao_bao_merged_profit.htm

  • 请求参数
    名称 类型 是否必须 默认值 说明
    token string 是 身份识别信息,参考 "获得用户Token接口"
  • 响应结果
    名称 类型 一定存在 示例 说明
    code string 是 SUCCESS SUCCESS : 获得用户成功
    (只有这个值,才有其它内容)
    TOKEN_ERROR : token无效
    dayProfit money 是 2.50 日收益
    principal money 是 10000.00 本金
    totalProfit money 是 5.00 当期收益
    totalMoney money 是 10005.00 当期收益+本金
    withdrawProfit money 是 143.66 已取现收益 _version=2
  • 示例
    {
    "principal": "2000.00",
    "code": "SUCCESS",
    "totalProfit": "0.49",
    "totalMoney": "2000.00",
    "dayProfit": "0.49"
}
